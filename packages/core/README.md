# @tweeq/core

Framework-neutral value semantics, validation, formatting, geometry, theme
generation, and state transitions shared by Tweeq's React and Vue renderers.

```sh
npm install @tweeq/core
```

Most applications should install `@tweeq/react` or `@tweeq/vue` instead; each
renderer installs core transitively. This package has no browser or framework
side effects and is safe to import during SSR.

The package name remains private until the repository publishing ADR's npm
ownership requirements are satisfied.
